Attacks the target, dealing the specified amount of damage. This method does not have a built-in cooldown.

Parameters
amountThe amount of damage to deal.
targetThe object to attack.
cooldownThe cooldown time for the attack. This parameter is included for consistency with the AttackWithCooldown method, but it has no effect in this method.
attackerTypeThe type of the attacker. This can be used to determine whether the attack is successful, or to apply type-specific effects.

Attacks the target, dealing the specified amount of damage. This method has a built-in cooldown.

Parameters
amountThe amount of damage to deal.
targetThe object to attack.
cooldownThe cooldown time for the attack. The object will be unable to attack again until this time has elapsed.
attackerTypeThe type of the attacker. This can be used to determine whether the attack is successful, or to apply type-specific effects.
Returns
An IEnumerator that can be used to track the progress of the cooldown.

Implemented in Supinfo.Project.Projectiles.Scripts.ProjectileAttack, and Supinfo.Project.Unit.Scripts.UnitCollision.UnitAttack.
